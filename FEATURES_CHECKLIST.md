# قائمة المهام المطلوبة (Checklist)

## أولاً: حساب العميل (Customer Dashboard) 🔒
- [x] **سجل الطلبات**: عرض قائمة الطلبات السابقة مع تفاصيل المنتجات، الحالة، والتكلفة. ✅ يعمل في `/orders` و `/orders/:id`
- [x] **المحفظة الإلكترونية (Wallet)**: نظام رصيد داخلي للعميل يمكن استخدامه في الشراء أو استرداد الأموال. ✅ يعمل في لوحة التحكم `/dashboard` مع عرض الرصيد والمعاملات
- [x] **الفواتير الضريبية (PDF)**: إمكانية تحميل فاتورة ضريبية رسمية بصيغة PDF لكل طلب مكتمل. ✅ يعمل في `/profile/invoices` (طباعة/تحميل)
- [x] **العناوين المحفوظة**: إدارة عناوين الشحن المتعددة لتسريع عملية الدفع. ✅ يعمل في `/dashboard` و `/profile`

## ثانياً: لوحة تحكم المسؤول (Admin Dashboard) 🔒
- [x] **إحصائيات المبيعات**: عرض إجمالي المبيعات مع فلاتر زمنية (يومي، أسبوعي، شهري). ✅ يعمل في لوحة الأدمن مع 4 فترات زمنية
- [x] **إدارة الطلبات**: تتبع عدد الطلبات الجديدة، قيد التنفيذ، والمكتملة. ✅ يعمل في لوحة الأدمن
- [x] **المنتجات الأكثر مبيعاً**: تحليل للمنتجات الأكثر طلباً لتحسين المخزون. ✅ يعمل في لوحة الأدمن
- [x] **صافي الربح**: حساب الأرباح بعد خصم التكاليف بناءً على البيانات المحاسبية. ✅ يُحسب الآن من تكاليف المنتجات الحقيقية

## ملاحظات تقنية:
- صافي الربح يُحسب من: `إجمالي الطلب - مجموع تكاليف المنتجات × الكميات - تكلفة الشحن`
- تكلفة كل منتج تُأخذ من حقل `cost` في المتغير أو المنتج الأساسي
- الإحصائيات الزمنية: اليوم / الأسبوع (يبدأ الأحد) / الشهر / كل الوقت
